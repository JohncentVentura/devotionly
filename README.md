# Devotionly
A digital devotion app for writing, organizing, and tracking on personal bible devotions.

## Tools
Next.js
* npx create-next-app@latest
Typescript
Tailwind CSS
Shadcn UI
* npx shadcn@latest init
* npx shadcn@latest add button
* npm install next-themes
* npx shadcn@latest add https://shadcnui-blocks.com/r/hero-05.json
* npx shadcn@latest add https://www.shadcnui-blocks.com/r/table-01.json
* npx shadcn@latest add popover
* npx shadcn@latest add command
Neon
* https://neon.com/
* npm install @neondatabase/serverless
Prisma
* npx prisma init
* npm install --save-dev prisma dotenv 
* npm i @prisma/client
* npx prisma generate
* npx prisma db push
*
* npx prisma migrate dev --name init
Stack Auth
* npx @stackframe/init-stack . --no-browser
